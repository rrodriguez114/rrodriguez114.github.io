The Web Design DeCal is an entirely student run decal that teaches web design and development principles to approximately 100 students at UC Berkeley. After enrolling in the class as a student in the Spring 2017 semester, I joined the team as a TA in the following semester. The 12 week, 2 and half hour per week crash course on web design looks to teach fundamental web design concepts and how to implement them using html, css, and javascript.

This semester, we worked to update the curriculum and teaching methods employed to better align the coding methods taught with design methods necessary to create a user friendly website. In order to effectively teach the wide range of topics covered in the class, I’ve challenged my own understanding of the design principles behind web design, become more familiar with front-end development, and learned to communicate effectively in both group settings and large audiences.
